## 🤝 参与贡献

### 如何选择任务
1. 查看[待办事项](./todo-list.md)
2. 选择合适优先级的任务
3. 创建 Issue 讨论
4. 开始开发

### 任务优先级
- 🔴 **高优先级** - 紧急重要
- 🟡 **中优先级** - 重要不紧急
- 🟢 **低优先级** - 锦上添花

### 开发流程
1. 认领任务
2. 设计方案
3. 编码实现
4. 测试验证
5. 代码审查
6. 合并发布
