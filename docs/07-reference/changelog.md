### 新增

### 改进

## [1.5.0] - 2025-01-30

### 新增
- 🎉 PWA 支持，可安装为桌面应用
- 📋 系统更新日志功能
- 🔄 完整的可视化工作流系统
- 📝 邮件模板系统 1.0

### 改进
- 修复系统工作流创建问题
- 优化我的工作流列表 UI

## [1.4.0] - 2025-01-20

### 新增
- 工作流引擎基础版本
- 34 种预定义节点类型
- 可视化流程编辑器
- 3 个系统预置工作流

### 改进
- 后端 API 性能优化
- 前端加载速度提升

## [1.3.0] - 2025-01-10

### 新增
- 邮件模板管理功能
- Markdown 编辑器支持
- 变量替换引擎
- 模板预览功能

### 修复
- 邮件发送附件丢失问题
- 文件夹排序错误
- 搜索结果分页问题

## [1.2.0] - 2024-12-20

### 新增
- 两步验证 (2FA) 支持
- 登录设备管理
- 会话管理功能
- 邮件追踪功能

### 改进
- 登录安全性增强
- UI 响应式优化
- 错误提示更友好

### 修复
- 密码重置邮件发送失败
- 时区显示错误

## [1.1.0] - 2024-12-01

### 新增
- 账号池功能
- 临时邮箱创建
- 验证码自动识别
- 订阅计费系统

### 改进
- 数据库查询优化
- 缓存策略改进
- UI 交互优化

### 安全
- 修复 XSS 漏洞
- 增强 CSRF 保护

## [1.0.0] - 2024-11-15

### 新增
- 🎉 首个正式版本发布！
- 完整的邮件收发功能
- 用户注册和登录系统
- 文件夹管理
- 邮件搜索
- 附件上传下载
- 深色/浅色主题切换
- 实时通知推送

### 已知问题
- 大附件上传可能超时
- 搜索功能仅支持基础查询
